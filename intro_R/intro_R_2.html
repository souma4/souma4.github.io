    Lesson 2: Fundamentals of Coding

    In this lesson, we will learn the basics of coding.

    Coding is the process of creating instructions for a computer to follow.

    These instructions are written in a programming language, such as R.

    Coding is used to create software, websites, and other digital products.

    Coding is an essential skill for anyone interested in technology.

    By the end of this lesson, you will be able to write simple programs in R.

    Let's get started!

    Basic Code

    In most conding languages, code is written in scripts.

    A script is a series of instructions that tell the computer what to do.

    Each instruction is called a statement.

    Statements are written in a specific order to create a program.

    Here is an example of a simple R script:

    # # marks comments

    x <- 5

    y <- 10

    z <- x + y

    print(z)

    # Output: 15

    In this script, we assign the the variable x a value of 5. Y is assigned 10, z is x +y, which is equivalent to 5+10.

    The print function displays the value of z.

    When you run this script, the output will be 15.

    This is a simple example of how code works.

    Math

    Coding is used to perform mathematical operations.

    Here are some basic math operations in R:

    # Addition

    x <- 5 + 3

    print(x)

    # Output: 8

    # Subtraction

    y <- 10 - 2

    print(y)

    # Output: 8

    # Multiplication

    z <- 4 * 2

    print(z)

    # Output: 8

    # Division

    a <- 16 / 2

    print(a)

    # Output: 8

    # Exponentiation

    b <- 2 ^ 3

    print(b)

    # Output: 8

    # Modulo/remainder

    b <- 15 %% 8

    print(b)

    # Output: 7

    with these you can now perform basic math!

    types

    In coding, there are several types of data that you can work with.

    Here are some of the most common data types in R:

    # Numeric- any number by default

    x <- 5

    # Character- wrapped in quotation marks

    y <- "Hello, World!"

    # Logical- TRUE or FALSE, boolean values

    while and for loops

    Loops are used to repeat a block of code multiple times.

    Here are two types of loops in R:

    # While loop

    x <- 1

    while (x <= 5) {

    print(x)

    x <- x + 1

    }

    # Output: 1 2 3 4 5

    # For loop

    for (i in 1:5) {

    print(i)

    }

    # Output: 1 2 3 4 5

    These loops are used to repeat a block of code multiple times.

    The while loop repeats the code block as long as the condition is true.

    The for loop repeats the code block for a specific number of times., the 1:5 creates a vector (a list of values) from 1:5. Try it!

    the loop iterates through that list 1:5. so lets try something new

    # For loop

    for (i in c("apple", "banana", "cherry")) {

    print(i)

    }

    # Output: apple banana cherry

    This loop iterates through the list of fruits and prints each one.

    This is a simple example of how loops work in R.

    apply

    The apply function is used to apply a function to a matrix, array, or list. these are usually preferred in R because loops run slowly. These are known as "vectorized" functions

    Here is an example of how the apply function works:

    # Create a matrix

    x <- matrix(1:9, nrow = 3)

    # Output:

    # 1 4 7

    # 2 5 8

    # 3 6 9

    # Apply the sum function to each column

    apply(x, 2, sum) # 2 says to apply over rows

    # Output: 12 15 18

    # Apply the sum function to each row

    apply(x, 1, sum) # 1 says to perform over #columns

    # Output: 6 15 24

    This is a simple example of how the apply function works in R.

    The apply function is a powerful tool for working with matrices, arrays, and lists.

    Indexing

    Indexing is used to access specific elements in a data structure.

    R indexes by row first, then column, values starting at 1. (not all languages do this)

    Here are some examples of indexing in R:

    # Create a vector

    #the c() function does this

    x <- c(1, 2, 3, 4, 5)

    # Access the first element

    print(x[1])

    # Output: 1

    # Access the last element

    print(x[length(x)])

    # Output: 5

    # Access multiple elements

    print(x[c(1, 3, 5)])

    # Output: 1 3 5

    # create a matrix

    x <- matrix(1:9, nrow = 3)

    print(x)

    1 4 7

    2 5 8

    3 6 9

    # Access the element in the first row and second column

    print(x[1, 2])

    # Output: 4

    # Access the entire second row

    print(x[2, ])

    # Output: 2 5 8

    # Access the entire second column

    print(x[, 2])

    # Output: 4 5 6

    This is a simple example of how indexing works in R.

    Indexing is a powerful tool for working with data structures in R.

    functions

    we've already used some functions, but remember functions are the bread and butter of coding as they perform specific tasks given specific inputs

    We can also create our own custom functions to perform specific tasks

    Try to do this whenenver possible. This will help you build a code base of functions that you can reuse later

    Here are some examples of functions in R:

    # Create a function that adds two numbers

    add <- function(x, y) {

    return(x + y)

    }

    # Call the function

    print(add(5, 3))

    # Output: 8

    # Create a function that multiplies two numbers

    multiply <- function(x, y) {

    return(x * y)

    }

    # Call the function

    print(multiply(4, 2))

    # Output: 8

    This is a simple example of how functions work in R.

    functions are what you will be working with for most of our work. They are the verbs of computer programming
