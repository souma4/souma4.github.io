    Basic Data Manipulation

    What is data manipulation?

    Data manipulation is the process of changing data to make it easier to read or to prepare it for analysis.

    Why is data manipulation important?

    Data manipulation is important because it allows you to clean and transform data so that it is in a format that is easier to work with. Additionally, it allows for quick, reliable, and transferable data analysis

    How to manipulate data in R

    In R, there are several packages that can be used to manipulate data. Some of the most common packages include dplyr, tidyr, and data.table. These packages allow you to filter, sort, group, and summarize data.

    we will use "tidyverse" which loads a suite of useful data wrangling packages, such as dplyr and tidyr

    To install the tidyverse package, you will need to run the following code in RStudio:

    install.packages("tidyverse")
    library(tidyverse)

    Once you have installed the tidyverse package, you can use the functions in the package to manipulate data.

    How to manipulate data

    Data classes

    In the last lesson we learned multiple data classes, such as vectors, briefly.

    In R, there are several data types that you will encounter when working with data. Some of the most common data types include:

    • Vectors: A vector is a one-dimensional array that can hold numeric, character, or logical data.
    • Matrices: A matrix is a two-dimensional array that can hold numeric, character, or logical data.
    • Data frames: A data frame is a two-dimensional array that can hold numeric, character, or logical data.
    • Lists: A list is a collection of objects that can hold numeric, character, or logical data.

    In R, you can use the str() function to determine the data type of an object.

    For example, if you have a data frame, you can use the following code to determine the data type of the object. We will use the Iris dataset for shared shared standard dataset:

    class(iris)
    # [1] "data.frame"

    In this example, the class() function returns "data.frame", which indicates that the object is a data frame.

    We can index this dataframe with the methods discussed last lesson, but we can also use the dplyr package to manipulate the data.

    For example, you can use the filter() function to filter rows based on a condition and select() which selects a column.

    Let's see what this looks like in code

    iris %>%
    filter(Species == "setosa") %>%
    select(Sepal.Length, Sepal.Width) %>%
    head()
    This code takes iris, filters the rows where the species is setosa, selects the Sepal.Length and Sepal.Width columns, and then displays the first few rows of the resulting data frame.

    %>% is the "pipe" function, which pipes the object preceding into the first argument of the following function. An alternative pipe is |>

    an example of pipe

    2 %>%
    sqrt()
    $ [1] 1.414214

    This code takes 2, pipes it into the sqrt() function, and then returns the square root of 2.

    This is a basic example of data manipulation in R. There are many other functions that you can use to manipulate data, and I encourage you to explore the dplyr and tidyr packages to learn more.

    Another important function is mutate(). mutate() allows you to create new columns based on existing columns.

    For example, you can use the following code to create a new column called "Petal.Ratio" that is the ratio of Petal.Length to Petal.Width:

    iris %>%
    mutate(Petal.Ratio = Petal.Length / Petal.Width) %>%
    head()

    This code creates a new column called "Petal.Ratio" that is the ratio of Petal.Length to Petal.Width and then displays the first few rows of the resulting data frame.

    Split-Apply-Combine

    The split-apply-combine strategy is a common data manipulation strategy that involves splitting the data into groups, applying a function to each group, and then combining the results.

    In R, you can use the group_by() and summarize() functions to implement the split-apply-combine strategy.

    For example, you can use the following code to calculate the mean Sepal.Length for each species in the Iris dataset:

    iris %>%
    group_by(Species) %>%
    summarize(mean(Sepal.Length))

    This code groups the data by the Species column, calculates the mean Sepal.Length for each group, and then displays the results.

    This is a basic example of the split-apply-combine strategy in R. There are many other functions that you can use to implement this strategy, and I encourage you to explore the dplyr and tidyr packages to learn more.

    Pivot_longer

    The pivot_longer() function is a useful function for reshaping data from wide to long format, which long format makes analysis very easy.

    For example, you can use the following code to reshape the Iris dataset from wide to long format:

    iris %>%
    pivot_longer(cols = c(Sepal.Length, Sepal.Width, Petal.Length, Petal.Width), names_to = "Measurement", values_to = "Value") %>%
    head()

    This code reshapes the Iris dataset from wide to long format, where the columns Sepal.Length, Sepal.Width, Petal.Length, and Petal.Width are reshaped into two columns: Measurement and Value.

    This shape enables you to group by the measurement type, which could be useful for building relationships between the variables.

    This is a basic example of reshaping data in R. There are many other functions that you can use to reshape data, and I encourage you to explore the tidyr package to learn more.
