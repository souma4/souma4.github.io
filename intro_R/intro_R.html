    About this course

    This course is designed to teach you the basics of coding in R. We will cover the basics of coding, data manipulation, data visualization, and statistical analysis. The course is designed to be hands-on, so you will be coding along with me, and I will walk through code line by line. I will also highly reccommend testing as we go on.

    Course Objectives

    By the end of this course, you should be able to:

    • Understand the basics of coding in R, but any standard coding programming language (javascript python, etc) will be easier to learn after this course.
    • Understand how to manipulate data in R
    • Understand how to visualize data in R
    • Understand how to perform basic statistical analysis in R

    Course Structure

    The course will be structured as follows:

    • Week 1: Introduction to R and RStudio
    • Week 2: Data Types and Data Structures
    • Week 3: Data Manipulation
    • Week 4: Data Visualization
    • Week 5: Statistical Analysis

    Course Materials

    The course materials will be provided on the course website. You will need to download R and RStudio to follow along with the course. I will provide links to download these programs in the first week of class.

    Additionally, we will install packages in R as we go. Instructions will be provided as we go along.
