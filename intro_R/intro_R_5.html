    Basic Statistics in R

    What is statistics?

    Statistics is the science of collecting, analyzing, and interpreting data. It is used to make informed decisions in a wide range of fields, including business, science, and social sciences. Statistics can help us understand the world around us by providing insights into patterns and relationships in data.

    Why is statistics important?

    Statistics is important because it allows us to make sense of data and draw meaningful conclusions from it. By analyzing data statistically, we can identify trends, patterns, and relationships that may not be immediately apparent. Statistics also helps us make informed decisions by providing evidence to support our conclusions.

    How is statistics used in R?

    R is a powerful tool for statistical analysis and data visualization. It has a wide range of built-in functions and packages that make it easy to perform statistical tests, create visualizations, and analyze data. Some of the most commonly used statistical functions in R include mean(), median(), sd(), and cor(). R also has packages like ggplot2 and plotly that make it easy to create visualizations of data.

    Examples

    Here are some examples of basic statistics in R:

    • Calculating the mean of a vector: mean(c(1, 2, 3, 4, 5))
    • Calculating the median of a vector: median(c(1, 2, 3, 4, 5))
    • Calculating the standard deviation of a vector: sd(c(1, 2, 3, 4, 5))
    • Calculating the correlation between two vectors: cor(c(1, 2, 3, 4, 5), c(5, 4, 3, 2, 1))

    Tidy Statistics

    Tidy statistics is a concept that emphasizes the importance of organizing data in a tidy format before performing statistical analysis. Tidy data is data that is organized in a consistent format, with each variable in a separate column and each observation in a separate row. By organizing data in a tidy format, we can make it easier to perform statistical analysis and create visualizations.

    We already covered tidy data, but lets emphasize split-apply-combine

    Split-Apply-Combine

    Split-apply-combine is a strategy for data analysis that involves three steps: splitting the data into groups, applying a function to each group, and combining the results. This strategy is commonly used in R to perform statistical analysis on grouped data. The dplyr package in R provides functions like group_by() and summarize() that make it easy to implement the split-apply-combine strategy.

    Example

    Here is an example of how split-apply-combine works in R:

    • Split the data into groups based on a variable: data %>% group_by(variable)
    • Apply a function to each group: data %>% group_by(variable) %>% summarize(mean = mean(value))
    • Combine the results: data %>% group_by(variable) %>% summarize(mean = mean(value))

    Modeling

    Modeling is the process of creating a mathematical representation of a real-world process. In statistics, modeling is used to make predictions, estimate parameters, and test hypotheses. R has a wide range of functions and packages that make it easy to create models and perform statistical analysis. Some of the most commonly used modeling functions in R include lm(), glm(), and nls().

    Example

    Here is an example of how to create a linear regression model in R using a standard dataset:

    # Load the dataset

    data(mtcars)

    # Create a linear regression model

    model <- lm(mpg ~ wt, data = mtcars)

    # Print the model summary

    summary(model)

    # Plot the relationship

    plot(mpg ~ wt, data = mtcars)

    abline(model)

    As we can see, we built a relationship between vehicle weight and miles per gallon.

    With this, we could understand the coefficients of the model with summary() and then plot the relationship
